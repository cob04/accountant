# Personal expense management

- Calculate monthly and annual burn rate
- Generate runway forecasts
- Track savings by category
- Generate report


### Expense Data model
- Category: Food, Housing, Clothing, Utility, Transportation, Investment, Saving, Msc
- Recurrence: Daily, Monthly, Quarterly, Annual (we can generate this attribute from the data)
- Amount: In Kenyan shillings
- Memo: (optional) note on the expense (require memos for Msc)
- Payment method: Card, Cash, Mobile
- Expense date: Date (maybe also time) when the expense happened.
- TimeStamp: date & time when the record is entered.
